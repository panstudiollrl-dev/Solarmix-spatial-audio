# Security Policy

Solarmix is an experimental Unity audio project. Security reports are still welcome, especially when they concern unsafe file loading, build scripts, dependency handling, or distributable app artifacts.

## Reporting

Please open a private security advisory on GitHub when possible. If that is unavailable, open a minimal public issue that describes the affected area without including exploit details.

## Scope

Relevant areas include:

- dataset and asset loading paths;
- build automation;
- Unity package or plugin handling;
- generated mobile projects;
- web demo deployment if reintroduced.

Third-party SDK vulnerabilities should also be reported upstream to the relevant maintainers.
